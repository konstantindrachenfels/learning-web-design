How it works

How does it work

The principle is simple, calculate the user's blood alcohol level to ensure safe journeys. This calculation takes into account your physiological data that you must enter when you register on the application. The alcohol consumed will be calculated using a breathalyzer with your phone.

Read More

The Rides

After calculating your blood alcohol level, the application will suggest a way to move safely. This will include completely autonomous cars (owned by the user or made available by a partner), cars with drivers or the user's non-autonomous personal car if the user is able to drive.

Read More

User Experience

Based on the different feedback that each user will be able to provide after each use of the application, improvements will be constantly proposed. In addition, a data operating system will offer routes according to the choices made by the user.
